# PRD: FastAPI AI-Native Backend Template

## 개요

외주 프로젝트용 FastAPI 백엔드 템플릿. AI 에이전트 코딩에 최적화된 구조.

## 문제 정의

- 프로젝트마다 보일러플레이트 반복 작성
- AI 에이전트가 코드 흐름 파악하기 어려운 구조
- LangGraph 에이전트를 기존 아키텍처에 끼워 맞추기 어려움
- 응답 형식, 에러 처리 등 일관성 부재

## 목표

1. **즉시 개발 시작** - clone 후 바로 비즈니스 로직 작성
2. **AI 에이전트 친화적** - 한 파일만 보면 흐름 파악
3. **완전한  AI 에이전트(Claude)가 코드 수정/생성 시

## 성공 지표

- 새 프로젝트 셋업 시간: 2시간 → 10분
- AI 에이전트가 기능 추가 시 파일 탐색 횟수 최소화
- 응답 포맷 일관성 100%
- Compact 후 컨텍스트 복원율 100%
